What is a FPS Booster and a Ping Reducer?

It is a Python-based optimization tool designed to improve the performance of games like Fortnite on Windows. It performs several actions like terminating unnecessary background processes, clearing DNS caches, and optimizing system settings.

Key Features

  • Terminate Chrome processes
  • Disable Visual Effects
  • Change DNS settings
  • Clear DNS Cache
  • Modify Hosts File
  • Disable Telemetry
  • Disable Windows Defender
  • Clean Temporary Files
  • Backup Important Files

Open Source & Safe

This tool is completely open-source, meaning anyone can inspect the code and verify that there are no viruses or malicious code. You can check the repository for the source code and contribute if you'd like!
